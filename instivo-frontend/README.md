## 💻 How to Run

First, run the development server:

```bash
# Open up terminal and run
$ npm install
$ npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the
result.

## 🎯 How to Test

```bash
# Open up terminal and run
$ npm run test
```
